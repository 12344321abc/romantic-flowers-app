# 📋 План улучшений сайта "Romantic Flower Farm"

После изучения проекта, вот список предложений по улучшению, организованный по категориям и приоритетам.

---

## 🎨 UI/UX улучшения

### Высокий приоритет
- [ ] **Добавить загрузочные индикаторы (спиннеры)** — при загрузке каталога, корзины, заказов пользователь не видит, что данные загружаются
- [ ] **Улучшить пустые состояния** — добавить красивые иллюстрации для пустой корзины, отсутствия заказов
- [ ] **Цветовая кодировка статусов заказов** — визуально различать "новый", "в работе", "завершён"

### Средний приоритет
- [ ] **Добавить фильтрацию и сортировку в каталоге** — по цене, дате добавления, наличию
- [ ] **Улучшить галерею изображений** — просмотр фото в полном размере (lightbox)
- [ ] **Анимации переходов** — плавное появление карточек товаров
- [ ] **Индикатор количества в корзине** — badge на иконке корзины

### Низкий приоритет
- [ ] **Темная тема** — переключатель светлой/темной темы
- [ ] **Улучшить адаптивность** — тонкая настройка для планшетов

## ⚙️ Функциональные улучшения

### Высокий приоритет
- [ ] **Редактирование профиля клиента** — возможность изменить адрес, контактные данные
- [ ] **Управление статусами заказов в админке** — смена статуса: новый → в работе → завершён/отменён
- [ ] **Пагинация списков** — разбиение на страницы для товаров и заказов
- [ ] **Валидация форм** — проверка данных на клиенте перед отправкой

### Средний приоритет
- [ ] **Категории цветов** — группировка по типам (розы, тюльпаны, пионы и т.д.)
- [ ] **Избранное** — возможность отметить любимые сорта
- [ ] **История изменения цен** — отслеживание изменений

### Низкий приоритет
- [ ] **Экспорт заказов в Excel/CSV** — для бухгалтерии
- [ ] **Напоминания о брошенной корзине** — через Telegram бот
